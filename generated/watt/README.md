## Architecture

Platformatic Watt application with 7 microservices:

| Service | Description |
|---------|-------------|
| `db` | Shared SQLite database with 21 migrations |
| `organizations` | Multi-tenant org management, auth, invites |
| `telemetry` | OTel ingestion, AI usage tracking |
| `alerts` | Alert rules and notifications |
| `solutions` | Solution Builder — create and publish AI solutions |
| `agent-gateway` | Desktop agent communication |
| `slack-bot` | Slack channel integration |
| `whatsapp-bot` | WhatsApp channel integration |
